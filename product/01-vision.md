# Product Vision

## Project Name
(Working Title) — Client Workflow Platform for Photographer

## Core Purpose
Design and build a lightweight booking and delivery system that replaces fragmented manual workflows with a structured, secure, and maintainable platform.

## Primary Goals
- Centralize booking requests
- Deliver password-protected client galleries
- Provide a minimal, cinematic portfolio experience
- Ensure fast image performance
- Keep administration simple for a non-technical user

## Long-Term Consideration
Although v1 supports a single photographer, the architecture should allow future expansion to multiple users without restructuring the database schema.
